# R6.시골지역에 맞는 네트워크 접속기술
## **기술 옵션 비교**

| 기술 | 장점 | 단점 | 적합한 상황 |
|------|------|------|------------|
| **위성 인터넷 (Starlink 등)** | - 빠른 구축<br>- 전국 커버리지<br>- 고속(50~200Mbps) | - 고가(장비비+월요금)<br>- 높은 지연시간(게임/화상회의 부적합)<br>- 기상 영향 | - 다른 옵션이 없는 외딴 지역<br>- 임시/신속 구축 필요<br>- 예산 충분 |
| **무선 광대역 (Fixed Wireless)** | - 비교적 빠른 구축<br>- 중간 속도(20~100Mbps)<br>- 중간 비용 | - 기지국 시야(LOS) 필요<br>- 장애물 영향 큼<br>- 대역폭 제한 | - 기지국이 가시권에 있는 지역<br>- 중간 인구밀도<br>- 지형이 평탄한 지역 |
| **모바일 네트워크 (4G/5G)** | - 이미 인프라 존재<br>- 즉시 사용 가능<br>- 이동성 | - 컨버전스 병목<br>- 데이터 한도/비싼 요금<br>- 음영 지역 문제 | - 모바일 인프라가 좋은 지역<br>- 저밀도/임시 사용<br>- 보조 연결용 |
| **DSL (전화선)** | - 기존 전화망 활용<br>- 낮은 구축비<br>- 안정적 | - 거리 제한(5km 이내)<br>- 속도 느림(<50Mbps)<br>- 케이블 노후 | - 전화국과 가까운 마을<br>- 낮은 대역폭 요구<br>- 유선망 확장 기반 |
| **FTTH (광케이블)** | - 초고속/저지연<br>- 안정성 최고<br>- 확장성 좋음 | - **구축 비용 매우 높음**<br>- 공사 기간 길다<br>- 투자 회수 어려움 | - 정부/지자체 지원 있는 경우<br>- 마을 단체 공동 투자<br>- 장기적 계획 가능 |
| **TV 케이블 (DOCSIS)** | - 기존 케이블망 활용<br>- 중고속 가능 | - 케이블망 없는 시골 많음<br>- 노드 공유로 속도 저하 | - 케이블방송 인프라 존재 지역 |

---

## **추천 접근 방식 (단계별)**

### **1단계: 현재 인프라 평가**
```
가장 저렴한 옵션 = 기존 인프라 활용
```
- **전화선 있는가?** → DSL 업그레이드 (VDSL2)
- **케이블TV 있는가?** → DOCSIS 3.1 활용
- **가까운 기지국 있는가?** → Fixed Wireless 검토
- **통신사 4G/LTE 신호 강한가?** → LTE 라우터 + 무제한 요금제

### **2단계: 지형적 조건 확인**
- **평탄한 농촌** → Fixed Wireless (타워 설치로 넓은 커버리지)
- **구릉지/산간** → 위성 인터넷 (지형 영향 없음)
- **집들이 흩어져 있음** → 위성 or 개별 LTE (유선 구축비 과다)

### **3단계: 예산과 요구사항**
- **저예산 + 기본 검색/동영상** → LTE/DSL
- **중간예산 + 안정적 업무용** → Fixed Wireless
- **고예산 + 초고속 필요** → 위성(Starlink) or 공동 FTTH 분담

---

## **실제 사례별 추천**

### **1. 가장 보편적이고 실용적인 조합 (2024년 기준)**
```
**주 연결: Starlink (위성)**
**백업/보조: LTE 라우터**
```
- **이유**: Starlink가 시골 지역에서 가장 균일한 성능을 보장하며, 가격도 점점 낮아지고 있음

### **2. 공동체 솔루션 (마을 단위)**
```
**마을 공동 FTTH 프로젝트**
1. 마을회관에 메인 광케이블 연결 (정부 지원사업 활용)
2. 마을 내부는 무선 링크(Fixed Wireless)로 배포
3. 개별 가정은 무선 CPE 설치
```
- **장점**: 장기적으로 가장 경제적이고 고품질
- **예시**: 한국 농어촌광역정보화사업, 미국의 ELOC(End of Life) 지원 프로그램

### **3. 저비용 긴급 솔루션**
```
**LTE/5G 홈 인터넷**
- 통신사의 "시골 특화 요금제" 확인 (데이터 무제한 or 저렴)
- 외부 안테나 설치로 신호 향상
```

## **한국 상황 고려 시 특이점**

### **국내 지원 정책 활용**
1. **농어촌 광역정보화 사업** (과학기술정보통신부)
   - 일부 비용 지원 가능
2. **지역별 통신사 의무 보급** 
   - 공공기관 위주이지만 주변 민가 확장 가능
3. **스마트팜 지원 사업** 
   - 농업용으로 신청하면 인터넷 인프라 지원 받을 수 있음

### **국내 통신사 옵션**
- **KT**: 윈드(Wind, Fixed Wireless) 서비스 - 시골 지역 특화
- **SKT/LGU+**: LTE/5G Home - 특정 지역 한정 무제한 요금제
- **스타링크**: 국내 공식 서비스 시작 (2022~)

## **결론: 단계적 결정 가이드**

### **질문에 답해보세요:**
1. **예산이 얼마인가요?**
   - 5만 원/월 이하 → LTE/DSL
   - 10만 원/월 이상 → Starlink
   - 일시 투자 가능 → FTTH 공동 구축

2. **어떤 용도인가요?**
   - 웹서핑/동영상 → LTE/Fixed Wireless
   - 재택근무/화상회의 → Starlink (저지연 위성)
   - 게임/실시간 스트리밍 → 가능하면 유선(FTTH)

3. **몇 가구인가요?**
   - 1~2가구 → 개별 해결 (위성/LTE)
   - 10가구 이상 → 공동 구축 (FTTH + 무선 배포)

### **최종 권장:**
**2024년 현재, 지형과 거주 환경이 매우 불리한 시골 지역에서는 *Starlink 등 저궤도 위성 인터넷(LEO)이 가장 실용적인 단일 솔루션*이다.**      
구축 시간이 거의 없고, 전국 어디서나 동일 품질을 기대할 수 있으며, 가격도 점점 대중화되고 있다.

하지만 **장기적으로 마을 공동체가 형성되어 있다면, FTTH 공동 구축 + 무선 배포 방식이 가장 경제적이고 지속 가능한 해결책**이 될 것이다. 정부 지원사업과 결합하면 더욱 효과적이다.    

시골(농촌·산간 지역)에서 인터넷 접속을 제공할 때는 지형·거리·인구 밀도 때문에 도시와 다른 기술을 사용해야 한다.          
일반적으로 가장 적절한 네트워크 접속 기술은 다음 순서로 선택된다.      

---

#  1순위: 무선 광대역(고정형 무선, Fixed Wireless Access: FWA)

시골에서 가장 많이 사용되는 방식이다.

### 이유

 광케이블을 집집마다 매설하기 어렵고 비용이 크다.
 전파로 수 km 이상 전달 가능하다.
 기지국 하나로 넓은 면적 커버 가능하다.

### 예

 LTE/5G 기반 FWA(집에 5G 공유기 설치하는 방식)
 와이브로 형태의 지역 무선망
 2.4GHz/5GHz 고정 무선 링크

---

#  2순위: 위성 인터넷(Satellite Internet)

지형이 매우 험하거나 인구가 너무 적은 지역에서 사용된다.

### 장점

 어디든 설치 가능
 초광역 커버리지

### 단점

 지연(latency)이 크고 비용이 높다
 날씨 영향 받을 수 있음
 스타링크 같은 LEO 위성은 지연이 낮지만 장비비는 높음

---

#  3순위: 광섬유 기반 FTTH(가능할 때만)

시골이라도 정부 사업(농어촌 광대역망)으로 광케이블이 들어온 경우는 최상.

### 장점

 안정적이며 품질 최고
 속도 일정

### 단점

 설치 비용이 매우 높아 지역 넓으면 비효율적이다.

---

#  결론

가장 적절한 기술: ‘무선 기반(Fixed Wireless Access, LTE/5G FWA)’
→ 비용 대비 효율, 커버리지, 설치 난이도 면에서 시골 지역에 최적이다.
