# Tit-for-Tat과 P2P 네트워크 정리


## 1. Tit-for-Tat 개념

**Tit-for-tat**은 P2P 네트워크에서 사용되는 교환 전략이다.        
다른 피어가 제공한 만큼 데이터를 제공하는 방식으로, 상호 이득을 기반으로 한 공정한 자원 교환을 유도한다.

### 특징

* 상대방이 제공한 만큼의 자원을 제공한다.
* 업로드에 적극적으로 참여한 피어에게 더 빠른 다운로드 속도를 제공한다.
* 무임승차를 억제하고 전체 네트워크 효율을 유지한다.

### 동작 방식

1. 피어가 다른 피어에게 데이터를 업로드한다.
2. 상대 피어는 그 기여도만큼 다시 데이터를 제공한다.
3. 교환이 지속되면서 서로 자원을 주고받는 구조가 형성된다.

---

## 2. P2P 네트워크 개념

**P2P(Peer-to-Peer) 네트워크**는 중앙 서버 없이 사용자가 직접 서로 데이터를 주고받는 네트워크 구조이다.

### 특징

* 중앙 서버 의존도가 낮다.
* 사용자(피어)가 동시에 공급자이자 소비자 역할을 한다.
* 사용자가 많아질수록 대역폭과 전송 속도가 증가할 수 있다.

### 동작 방식

1. 사용자가 특정 파일의 조각을 요청한다.
2. 이미 해당 조각을 가진 다른 피어들로부터 데이터를 직접 다운로드한다.
3. 다운로드한 조각은 다른 피어들에게 제공할 수 있다.

---

## 3. Tit-for-Tat과 P2P 관계

Tit-for-tat은 P2P 네트워크에서 자원 공유를 효율적으로 유지하기 위한 알고리즘이다. 즉, P2P는 구조이고 tit-for-tat은 그 안에서 사용되는 전략이다.

---
## 4. 비교 표

| 구분    | Tit-for-Tat                | P2P 네트워크                   |
| ----- | -------------------------- | -------------------------- |
| 정의    | 피어 간 데이터 교환 전략             | 중앙 서버 없이 피어 간 직접 데이터 공유 구조 |
| 목적    | 공정한 자원 분배, 무임승차 방지         | 확장성 높은 데이터 전송 구조 제공        |
| 동작 방식 | 상대가 준 만큼 되돌려주는 방식으로 데이터 교환 | 피어 간 직접 통신을 통해 데이터 조각 교환   |
| 역할    | 인센티브 메커니즘                  | 전체 데이터 전송 시스템              |
| 적용 여부 | 토렌트 등에서 사용                 | 다양한 파일 공유 시스템에서 사용         |

---

## 6. 정리

* P2P는 피어들이 직접 데이터를 주고받는 구조이다.
* Tit-for-tat은 P2P 네트워크에서 공정성과 효율을 높이기 위해 사용하는 전략이다.
* 두 개념은 서로 보완적이며, tit-for-tat은 P2P의 핵심 알고리즘으로 널리 사용된다.
