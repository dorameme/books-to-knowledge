

## 1. PAN (Personal Area Network)

### 개념 정리

PAN은 **개인 주변의 매우 짧은 거리에서 장치 간 통신을 수행하는 네트워크**이다.        
통신 범위는 일반적으로 수 미터에서 수십 미터 수준이다.      
저전력, 근거리, 소형 기기 연결이 목적이다.

---

### 예시

* 스마트폰과 무선 이어폰 연결이다.
* 노트북과 무선 마우스 연결이다.
* 스마트워치와 스마트폰 연결이다.

---

### PAN 요약 표

| 구분    | 설명                  |
| ----- | ------------------- |
| 범위    | 수 m ~ 수십 m이다        |
| 목적    | 개인 기기 연결이다          |
| 특징    | 저전력, 근거리이다          |
| 대표 기술 | Bluetooth, ZigBee이다 |

---

## 2. Bluetooth

### 개념 정리

Bluetooth는 **PAN 환경에서 사용되는 근거리 무선 통신 기술**이다.
저전력 통신을 지원하며, 주로 1:1 또는 소규모 연결에 사용된다.
2.4GHz ISM 대역을 사용한다.

---

### 특징

* 페어링 과정을 통해 장치를 연결한다.
* 주파수 도약 방식을 사용해 간섭을 줄인다.
* Bluetooth Low Energy는 매우 낮은 전력 소모를 목표로 한다.

---

### 예시

* 무선 이어폰이다.
* 차량용 핸즈프리이다.
* 스마트워치이다.

---

### Bluetooth 요약 표

| 구분      | 설명          |
| ------- | ----------- |
| 네트워크 유형 | PAN이다       |
| 통신 거리   | 약 10m 전후이다  |
| 전력 소모   | 낮다          |
| 주요 용도   | 오디오, 웨어러블이다 |

---

## 3. ZigBee

### 개념 정리

ZigBee는 **저전력·저속 데이터 전송을 위한 PAN 기반 무선 통신 기술**이다.
IEEE 802.15.4를 기반으로 한다.
센서 네트워크와 IoT 환경에 최적화되어 있다.

---

### 특징

* 메시 네트워크 구성이 가능하다.
* 배터리 수명이 매우 길다.
* 데이터 전송 속도는 낮다.

---

### 예시

* 스마트 홈 센서이다.
* 조명 제어 시스템이다.
* 산업용 센서 네트워크이다.

---

### ZigBee 요약 표

| 구분    | 설명              |
| ----- | --------------- |
| 기반 표준 | IEEE 802.15.4이다 |
| 전송 속도 | 낮다              |
| 전력 소모 | 매우 낮다           |
| 네트워크  | 메시 구조이다         |

---

## 4. Bluetooth와 ZigBee 비교

| 구분      | Bluetooth   | ZigBee       |
| ------- | ----------- | ------------ |
| 주요 목적   | 개인 기기 연결이다  | 센서 네트워크이다    |
| 데이터 속도  | 상대적으로 높다    | 낮다           |
| 전력 소모   | 낮다          | 매우 낮다        |
| 네트워크 구조 | 스타 구조이다     | 메시 구조이다      |
| 사용 사례   | 이어폰, 웨어러블이다 | IoT, 스마트 홈이다 |

---

Bluetooth와 ZigBee는 모두 PAN 응용 기술이다.      
ZigBee는 Bluetooth보다 더 저전력·저비용을 목표로 설계된 기술이다.

## 핵심 한 줄 요약

* PAN은 개인 주변의 근거리 네트워크 개념이다.
* Bluetooth는 개인 기기 연결에 최적화된 PAN 기술이다.
* ZigBee는 대규모 저전력 센서 네트워크를 위한 PAN 기술이다.
