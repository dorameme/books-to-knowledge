AON과 PON은 광 통신 기반의 액세스 네트워크 방식을 말한다.      
둘 다 인터넷을 건물까지 전달할 때 쓰는 구조지만, 방식이 완전히 다르다.

#### (참고로 에이-오-엔 , 피-오-엔 이라고 읽음)
---

# AON(Active Optical Network)

전기가 필요한 장비(스위치, 라우터)가 중간에 개입하는 방식이다.

## 핵심 특징

* 중간 구간에 스위치·라우터 같은 액티브 장비가 있다.
* 각 가입자에게 독립된 광 회선 또는 VLAN을 제공한다.
* 거리 제약이 적고, 대역폭을 더 쉽게 확장할 수 있다.
* 구축비용이 높다(중간 장비 필요, 유지관리 필요).

## 구조 예시

```
ISP ---- 스위치(전원 필요) ---- 건물 ---- 사용자
```

## 장점

* 트래픽 충돌 없음 → 속도 보장 쉽다.
* 장애 구간 파악과 관리가 쉽다.
* 확장성과 품질이 높다.

## 단점

* 장비가 많아져 비용 상승.
* 장애 가능 포인트가 늘어난다.

---

#  PON(Passive Optical Network)

중간에 전원이 필요한 장비가 없는 '수동형' 광 네트워크 방식이다.

## 핵심 특징

* 중간엔 Splitter(광 분배기)만 존재 → 전원이 필요 없다.
* 하나의 광 회선을 여러 가입자가 공유한다.
* 구축비용이 저렴하고 대량 구축에 유리하다.
* FTTH(가정까지 광)에서 주로 사용된다.

## 구조 예시

```
ISP ---- Splitter(전원 없음) ---- 여러 사용자
```

## 장점

* 설치비가 매우 저렴하다.
* 장비가 적어 유지비가 낮다.
* 대규모 구축 시 경제적이다.

## 단점

* 공유 구조라서 속도 보장이 어렵다.
* Splitter로 인해 거리가 제한된다.
* 장애 지점이 추적하기 어렵다.

---

#  AON vs PON 한눈에 비교

| 구분    | AON                     | PON                      |
| ----- | ----------------------- | ------------------------ |
| 중간 장비 | 스위치/라우터 등 액티브 장비 필요 | Splitter만 사용(전원 불필요) |
| 구성 형태 | 점대점(Point-to-Point)     | 포인트-투-멀티포인트(P2MP)        |
| 속도 보장 | 뛰어남                     | 공유라서 제한적                 |
| 구축 비용 | 높음                      | 매우 낮음                    |
| 유지보수  | 복잡                      | 단순                       |
| 사용 분야 | 기업망, 고품질 네트워크           | 일반 가정 FTTH               |

---

#  결론

* PON: 비용 아끼고 대량 구축하려고 사용하는 가정용 광 네트워크 방식
* AON: 기업이나 고품질 서비스를 위해 속도·관리성을 높이는 방식
